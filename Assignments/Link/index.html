DOS Command

  • 1_If you want to open run box, ↦ Window + R
  • 2_To Enter DOS command line mode, ↦ Type "cmd" in the Run Box and press Enter

Basic CMD Commands

  1. To check your 'current directory', use cd + Enter
  2. To check your 'directory list' in current directory, use dir + Enter
  3. To change 'directory to upper level', use cd .. + Enter
  4. To change 'directory to another directory', use cd folder_name + Enter
  5. To change 'directory to root', use cd\ + Enter
  6. To 'create a directory', use mkdir directory_name + Enter
  7. To 'create a file using notepad', use notepad file_name.txt + Enter
  8. Show texts in file, use type file_name + Enter
  9. Move file to directory, use move file_name directory_name
  10. To 'delete a file', use del file_name
